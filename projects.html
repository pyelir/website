This applet provides a visual representation of the Erdos-Renyi G(n,p) random graph model. To do this, it first generates a representation of the graph as a set of nodes and edges, with each edge assigned a probabilty uniformly at random. To display this abstract object, it first finds a minimum spanning tree (with edges weighted according to their assigned probabilities) using Prim's algorithm, and then feeds that MST into the Kamada-Kawai force-directed graph layout algorithm. This helps give a relatively clean and uncluttered view of the graph for low values of p.


Users can adjust the value of p using a slider and watch the graph evolve in real time to reflect the change in its parameters. Future modifications might include enhancements to better visualize zero-one laws (such as the emergence of a giant component) as the parameters shift, as well as a faster layout algorithm to improve performance.


The original idea for this project is due to Prof. Lenore Cowen at Tufts, who also suggested computing the layout using an MST. I worked with Michael Gold to implement an eariler version a class project, of which this is a complete re-wirte. In the version, we used a Jupyter notebook, which allowed us to use the NetworkX package to handle graph representation and layout, things which I re-implemented here myself.


Prof. Greg Aloupis provided help and guidance for a separate, later class project I did on force-directed graph layout algorithms, which helped greatly in writing the code for this applet.
